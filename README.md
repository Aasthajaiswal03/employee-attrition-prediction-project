📌 Problem Statement
Employee attrition is a major challenge for HR departments. This project builds a classification model to predict whether an employee is likely to leave the company, using the IBM HR Analytics Employee Attrition Dataset. <br>

Project Phases:<br>
 1  Data Loading — IBM HR Dataset from Kaggle <br>
 2  EDA — Distribution plots, correlation heatmap, feature analysis <br>
 3  Preprocessing — Label encoding, train-test split (80/20) <br>
 4  Model Building — Logistic Regression, Random Forest, XGBoost <br>
 5  Model Evaluation — Accuracy, Precision <br>
 6  Deployment — Streamlit app with interactive predictions <br>
